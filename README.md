## Struktur Proyek dan PIC

- `src/`
  - `main.c` — Abi
  - `arsip.c` — Nabilah
  - `animasi.c` — Arkaan
  - `skor.c` — Kimi
  - `sorting.c` — Diyar
  - `visual.c` — Nindira
  - `struktur.h` — Nabilah
- `data/` — berisi file `thadiah.txt` dan `tgerak.txt` untuk input/output game

## Cara Build

Gunakan compiler C seperti `gcc`.

```powershell
cd d:uas-algoritma-kelompok3
gcc src\*.c -o lite-o.exe
```

```powershell
.\lite-o.exe
```
